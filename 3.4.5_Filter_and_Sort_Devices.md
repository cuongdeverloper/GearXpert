# 3.4.5 Filter and Sort Devices

**Function trigger:** The User selects one or more filter options and/or sort options on the Device Listing page.

**Function Description:** This function allows the user to filter and sort the list of rental devices based on selected criteria such as category, price range, availability, location, rating, and sort order. No rental transaction or checkout is performed in this function.

**Data Processing:**
- The system retrieves the full list of devices from the database.
- The system applies the selected filter criteria to the device list.
- The system applies the selected sorting criteria to the filtered device list.
- The system updates the displayed device list based on the filtered and sorted results.
- No cart or order is created during this process.

**Screen layout:**
- Fig 3.4.14 Filter and Sort Devices Screen

**Function details:**

**Data provided by system:**
- Device name
- Device thumbnail image
- Rental price per day
- Device category
- Device location
- Device availability status (AVAILABLE / UNAVAILABLE)
- Average rating (if available)
- Device creation or listing date

**Customer inputs:**
- **Filter options:**
  - Select device category
  - Select price range (minPrice, maxPrice)
  - Select availability status (AVAILABLE / UNAVAILABLE)
  - Select location (city or warehouse)
  - Select minimum rating
- **Sort options:**
  - Sort by price (low → high / high → low)
  - Sort by average rating
  - Sort by newest devices
- **Actions:**
  - Apply filter and sort options
  - Clear filter and sort options

**Business rules:** BR-04, BR-05, BR-14

**Normal case:**
User accesses Device Listing page →
System displays full device list →
User selects filter and/or sort options →
System applies filters and sorting →
System displays filtered and sorted device list successfully.

**Abnormal cases:**
- No devices match the selected filters →
  Message: "No devices match your selected filters."
- Invalid filter values →
  Error message: "Invalid filter criteria selected."
- Invalid sort option selected →
  Error message: "Invalid sort option selected."
- System error →
  Error message: "Unable to filter and sort devices at the moment."
